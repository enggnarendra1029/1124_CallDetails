6. About (Short Description / Summary): Our project based on an implementation of existing android phone feature Call log.Call log

maintains a list of all types of incoming ,outgoing and missed calls made through the phone. All

the details of calls stored in in build database of android are accessed via content providers

.Details of all the calls are maintained in the call log until user deletes them himself.
